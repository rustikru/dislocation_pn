▶ create_info_modal_dialog Информационный / предупреждающий диалог Modal
Описание

Универсальный диалог для отображения сообщений, ошибок и предупреждений. Вызывается из большинства других форм при ошибках валидации или AJAX-ответах.

Параметры функции
ПараметрТипОписание
p_titlestringЗаголовок диалога
p_msgstring / HTMLТекст сообщения
Конфигурация диалога
ID элемента:#modalDialog
Ширина:400px
Modal / Resize:да / нет
Кнопки:«Закрыть»
AJAX-запросы

Нет.
