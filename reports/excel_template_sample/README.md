# Шаблон Excel-отчёта

Файлы:

- `template.xlsx` — Excel-шаблон с плейсхолдерами.
- `report.php` — пример формирования отчёта через PhpSpreadsheet.

Что менять в `report.php`:

- `$headerSql` — запрос для одиночных значений шапки.
- `$tableSql` — запрос для табличной части.
- `$placeholders` — какие метки в Excel заменить значениями из `$headerSql`.
- `$tableStartRow` — номер строки, которую нужно размножить под таблицу.
- `$tableColumns` — соответствие колонок Excel колонкам SQL-запроса.

Если нужны объединённые ячейки, лучше объединить их прямо в `template.xlsx`.
