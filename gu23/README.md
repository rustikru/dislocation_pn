# ГУ-23

Папка `gu23` содержит модуль актов общей формы ГУ-23.

## Основные файлы

- `index.php` - главная страница модуля. Проверяет авторизацию и доступ, подключает стили и JavaScript.
- `GuActRepository.php` - основной обработчик AJAX-действий модуля: акты, справочники, роли, файлы, согласование.
- `ApprovalRepository.php` - работа со страницей подписания по ссылке: акт, подписант, статус согласования.
- `approve.php` - отдельная страница для подписания или отклонения акта по HMAC-ссылке.
- `get_file.php` - отдача вложенного файла акта на просмотр или скачивание.
- `browser_check.php` - проверка браузера перед открытием модуля.
- `config.php` - настройки модуля, сейчас в основном режим отправки писем.
- `js.php` - сборка JS-файлов из `components` в один скрипт.
- `gu23.css` - стили модуля.
- `Gu23Logger.php` - запись ошибок и служебных сообщений в `gu23/log`.
- `session_debug.php` - локальная страница для просмотра сессии при отладке.

## Oracle package

- `XX_DISL_GU23_PKG.pks` - спецификация Oracle-пакета.
- `XX_DISL_GU23_PKG.pkb` - тело Oracle-пакета.

Эти файлы описывают серверную логику в БД: выборки, сохранение актов, роли, справочники, согласование.

## Frontend

Папка `components` содержит JS модуля.

- `app.js` - запуск приложения, загрузка справочников, переключение экранов.
- `api.js` - общий AJAX-вызов в `data.php`.
- `state.js` - общее состояние модуля: справочники, текущий экран, активный проект.
- `nav.js` - боковое меню.
- `registry.js` - журнал актов, фильтры, таблица актов.
- `form.js` - форма создания и редактирования акта.
- `card.js` - карточка акта: реквизиты, вагоны, подписанты, файлы, действия.
- `refs.js` - экран справочников.
- `roles.js` - экран ролей и полномочий.
- `wagonSearch.js` - поиск актов по номеру вагона.
- `ui.js` - общие элементы интерфейса: статусы, модальные окна, уведомления.
- `utils.js` - мелкие функции: даты, HTML-экранирование, разбор номеров вагонов.

## Подписание и письма

Папка `lib` содержит классы для ссылок и писем.

- `HmacApproval.php` - генерация и проверка HMAC-ссылок на подписание.
- `ApprovalMailer.php` - формирование HTML-письма и сохранение/отправка письма.

Папка `mail` используется в режиме сохранения писем в файл.

## Отчеты

Папка `report` содержит генерацию DOCX.

- `report.php` - endpoint для скачивания DOCX-отчета по акту.
- `GuActDocxReport.php` - сборка DOCX-файла из шаблона.
- `template/act23_start.docx` - шаблон акта начала простоя.
- `template/act23_end.docx` - шаблон акта окончания простоя.
- `template/act23_general.docx` - шаблон прочего акта.

## Служебные папки

- `log` - логи модуля.
- `mail` - сохраненные письма при режиме `send_file`.
- `request_data` - служебные данные запросов, если используются при отладке или импорте.
